# Hardhat Project

```shell
pnpm install
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help

# Pour le frontend
cd frontend
pnpm install
pnpm run start
```
